# Google Voice 中文注册、登录、转移和使用方法

## 注意事项

谷歌对新账户（一般随机号码是新账号，代注册、靓号、永久号除外）风控严格，对网络要求也高：
1. 优先用美国 IP 登录，IP 越纯净越好
2. 登录后不要频繁操作，过几小时修改密码和辅助邮箱
3. 前几天不要登陆多设备
